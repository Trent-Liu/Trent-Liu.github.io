C++自学——标准库类型String以及cctype头文件

本文最后更新于：2021年9月14日 下午

标准库类型String

学习C语言的时候，对于字符串的处理只能使用字符数组，十分麻烦，C++中有了标准库类型String，对于字符串的处理就简单了很多

使用前提

初始化字符串的方式

直接初始化和拷贝初始化

拷贝初始化，直接用=初始化，编译器将等号右侧的初始值拷贝到新创建的对象

直接初始化，不使用=

如果一定要用拷贝初始化，需要显式地创建一个临时对象用于拷贝

string的基本操作

  1. os << s

    将s写到输出流os中，返回os

  2. is >> s

    从is中读取字符串赋给s，字符串以空白分隔，返回is

  3. getline(is, s)

    从is中读取一行，赋给s，并返回is

  4. s.empty()

    s为空返回true，否则返回false

  5. s.size()

    返回s中字符的个数

  6. s[n]

    返回s中第n个字符的引用，位置n从0计起，类似C中的字符数组

  7. s1+s2

    返回s1和s2连接后的结果

  8. s1=s2

    用s2的副本代替s1中原来的字符

  9. s1==s2（s1！=s2）

    如果s1和s2中所含的字符完全一样，则它们相等，string对象的相等性判断，对字母的大小写敏感

  10. < <= > >=

    利用字符在字典中的顺序进行比较，且对字母的大小写敏感

因为在执行读取操作的时候，string会自动忽略开头的空白(空格符，换行符，制表符)，并从第一个真正的字符开始读起，直到遇见下一处空白为止。

因此若是输入” Hello World!”，结果还是Hello

若还是输入Hello World！

  • size函数返回的是一个string::size_type类型的值

    stirng类及其他的大多数标准库类型都定义了几种配套的类型，这些配套类型体现了标准库类型与机器无关的特性，size_type就是一种。

比如

s.size() < n;

n若是一个负值的int，则上述表达式就为true，负值n会自动地转换成一个较大的无符号值。
